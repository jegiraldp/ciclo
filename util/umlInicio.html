Lenguaje Unificado de Modelado - UML: "Es un lenguaje estándar para construir planos de software. Es un lenguaje gráfico gráfico para visualizar, especificar, construir y documentar los artefactos de un sistema con gran cantidad de software.(Tomado de: El lenguaje unificado de modelado. UML 2.0. Booch 2006).
Para comprender UML es necesario conocer los componentes principales del lenguaje, estos son: Los elementos básicos de contrucción, diagramas asociados a la estructura y diagramas de comportamiento o dinamismo.
Los diagramas presentados en esta sección están basados en una inmobiliria, donde se requiere un sistema software para el registro de propiedades (casas,apartamentos,etc).
