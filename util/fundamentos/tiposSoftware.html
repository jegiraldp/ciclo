Tipos de software Definición de software -- (Tomado de: Ingenieria del Software.Un Enfoque Practico, R.S Pressman)

  • Software de Sistemas: conjunto de programas escritos para dar servicio a otros programas. Ejemplo de este tipo de software son: compiladores, editores, herramientas para administrar archivos, componentes de sistemas operativos,software de redes, procesadores de telecomunicaciones, entre otros.


  • Software de aplicación:Son programas aislados que resuelven una necesidad específica de negocios. El software de aplicación se usa para controlar funciones de negocios en tiempo real (por ejemplo, procesamiento de transacciones en punto de venta, control de procesos de manufactura)


  • Software de ingenierías y ciencias:Son programas que realizan cálculos avanzados. Las aplicaciones van de la astronomía a la vulcanología, del análisis de tensiones en automóviles a la dinámica orbital del transbordador espacial, y de la biología molecular a la manufactura automatizada


  • Software incrustado:este software reside dentro de un producto o sistema y se usa para implementar y controlar características y funciones para el usuario final y para el sistema en sí. El software incrustado ejecuta funciones limitadas y particulares (por ejemplo, control del tablero de un horno de microondas) o provee una capacidad significativa de funcionamiento y control.


  • Software en la web:Este software se caracteriza porque es ejecutado y accedido a través de un navegador web. Su desarrollo ha sido con tecnologías web (HTML, CSS, Javascript, etc) Su principal ventaja está en la posibilidad de ser ejecutadas en varios disposivitos (móviles, escritorio). A esto se suma que el software para la web, es totalmente dependiente de la Internet, pues tiene una fuerte orientación a la conexión.


  • Software Inteligente:Este tipo de software aplica técnicas de la Inteligencia Artificial para mejorar sus resultados, procedimientos y rendimiento. Se caracteriza por el manejo de grandes cantidades y variedad de ellos, con diferentes complejidades y características. Las aplicaciones son diversas, entre ellas se tiene: Robótica, agentes de software, reconocimiento de voz, realidad virtual, sistemas de recomendación, búsqueda y recuperación de información, sistemas tutoriales inteligentes.
