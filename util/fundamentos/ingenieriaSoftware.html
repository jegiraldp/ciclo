Definición de Ingeniería de software -- Algunas definiciones

  • La Ingeniería de Software es: un proceso que consiste en un conjunto de actividades relacionadas entre si, para la transformación de requisitos de usuario a un producto software que resuelve un problema identificado. El proceso lo llevan a cabo los Ingenieros de software, que desarrollan, mantiene y operan el software. La Ingeniería de software comprende procedimientos y herramientas que soportan el diseño, construcción, pruebas y configuración del software
    (Definición tomada de: SWEBOK 2004, Software Engineering Body of Knowledge)


  • La Ingeniería de Software es: Es la aplicación de un proceso de de desarrollo de manera lógica, cuantificable, documentable para el desarrollo, mantenimiento y operación de todo tipo de software. Las características principales de un software son: abstracto e invisible, estático y dinámico, acorde a las organizaciones, posee un ciclo de vida y no pasa de moda.
    (Definición tomada de: Software Engineering 2014. ACM/IEEE)


  • La Ingeniería de Software es: una disciplina que incluye procesos, métodos y herramientas que permiten elaborar a tiempo y con calidad sistemas complejos basados en computadoras. El proceso de software incorpora cinco actividades estructurales: comunicación, planeación, modelado, construcción y despliegue que son aplicables a todos los proyectos de software.
    (Definición tomada de: Ingenieria del Software.Un Enfoque Practico, R.S Pressman)


  • La Ingeniería de Software es: una disciplina de ingeniería que comprende todos los aspectos de al producción de software, desde las etapas iniciales de la especificación del sistema, hasta el manteniemiento de éste después que se utiliza. La ingeniería de software no solo comprende los procesos técnicos del desarrollo de software, sino también actividades como gestión de proyectos de software. desarrollo de herramientas, métodos y teorías de apoyo.
    (Definición tomada de: Ingenieria del Software.I. Sommerville)
